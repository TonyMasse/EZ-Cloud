#### 01.single.json

Contains a single log
- URL: [https://tonymasse.github.io/EZ-Cloud/samples/01.single.json](https://tonymasse.github.io/EZ-Cloud/samples/01.single.json)

``` json
{
  "timestamp":"20210422T16:40:00",
  "destination":{
    "ip":"172.16.1.2",
    "port":443
  },
  "source":{
    "ip":"192.168.0.1",
    "port":44444
  }
}
```
